# Getting started with implement Typescript inside React project

This project was created by Create React App.

### What you can learn from this project

This project is a quick start project to show you how to use typescript in react at Props, state, event handler and Refs.

1. At src/props, you can learn how to use typescript to customize the shape of react component/
2. At src/state, you can learn how to tell typescript what type of value would be assign to the state.
3. At src/events, you can learn how to assign type in the event handler.
4. At src/refs, you can learn how to assign type to the refs which is refer to a HTML element.

### How to Run this project locally

`npm start`
Run this app in the development mode.
Open http://localhost:3000/ to view it in the browser.
